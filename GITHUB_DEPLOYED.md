## 🚀 克隆和使用

其他人可以通过以下步骤使用你的项目：

```bash
# 1. 克隆仓库
git clone https://github.com/xuldqi/xhs.git
cd xhs

# 2. 安装前端依赖
npm install

# 3. 安装后端依赖
cd backend
npm install

# 4. 配置环境变量
# 复制 .env.example 为 .env 并填入 API Key
cp .env.example .env
# 编辑 .env 文件，添加 DeepSeek API Key

# 5. 启动后端服务
npm run dev

# 6. 启动前端服务（新终端）
cd ..
npm run dev
```

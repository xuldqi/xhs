# 🖼️ 创建 OG Image 指南

## 什么是 OG Image？

OG Image 是当你的网站链接在社交媒体（微信、QQ、微博等）上分享时显示的预览图片。

## 📐 规格要求

- **尺寸**: 1200x630px（推荐）
- **格式**: PNG 或 JPG
- **文件名**: `og-image.png`
- **位置**: `public/og-image.png`

## 🎨 设计建议

### 内容要素
1. **主标题**: "小红书涨粉助手" (大字体)
2. **副标题**: "AI 智能生成 12 章节涨粉指南" (中等字体)
3. **特色功能**: "上传截图 → 智能分析 → 专业指南" (小字体)
4. **Logo/图标**: 小红书相关元素

### 设计风格
- **背景**: 渐变色（粉色到橙色，符合小红书风格）
- **字体**: 现代、清晰、易读
- **颜色**: 主色调使用小红书的红色 (#FF2442)
- **布局**: 左右布局或居中布局

## 🛠️ 制作工具

### 在线工具（推荐）
1. **Canva** (https://canva.com)
   - 搜索 "Facebook Post" 模板
   - 调整尺寸为 1200x630px
   - 免费，模板丰富

2. **Figma** (https://figma.com)
   - 专业设计工具
   - 免费版功能足够

3. **稿定设计** (https://gaoding.com)
   - 中文界面
   - 有小红书风格模板

## 📝 制作步骤（使用 Canva）

1. **创建设计**
   - 访问 https://canva.com
   - 点击 "创建设计"
   - 选择 "自定义尺寸" → 1200 x 630 像素

2. **设置背景**
   - 选择渐变背景
   - 颜色：从 #FF6B9D 到 #FF2442

3. **添加文字**
   ```
   主标题: 小红书涨粉助手
   字体: 思源黑体 Bold, 48px, 白色
   
   副标题: AI 智能生成专业涨粉指南
   字体: 思源黑体 Regular, 24px, 白色
   
   功能说明: 上传截图 → 智能分析 → 导出PDF
   字体: 思源黑体 Light, 16px, 白色/透明度80%
   ```

4. **下载**
   - 点击 "下载"
   - 选择 PNG 格式
   - 保存为 `og-image.png`

## 📂 文件放置

```bash
# 将下载的文件移动到 public 目录
mv ~/Downloads/your-og-image.png xiaohongshu-guide-generator/public/og-image.png
```

## ✅ 验证效果

### 本地测试
1. 确保文件在 `public/og-image.png`
2. 访问 http://localhost:5173
3. 查看页面源代码，确认有：
   ```html
   <meta property="og:image" content="/og-image.png">
   ```

### 在线测试（部署后）
1. **Facebook 分享调试器**
   - https://developers.facebook.com/tools/debug/
   - 输入你的网站 URL

2. **微信分享测试**
   - 在微信中发送网站链接
   - 查看预览卡片

## 🚨 注意事项

1. **文字大小**: 确保在小屏幕上也能清晰阅读
2. **对比度**: 文字与背景要有足够对比度
3. **品牌一致**: 颜色和风格要与网站保持一致
4. **文件大小**: 尽量控制在 300KB 以内

## 📱 不同平台效果

- **微信**: 显示为链接预览卡片
- **QQ**: 显示为分享卡片
- **微博**: 显示为链接预览
- **Facebook**: 显示为 Link Preview
- **Twitter**: 显示为 Twitter Card

## 🔧 配置步骤

### 1. 配置 Google Analytics

```bash
# 1. 复制环境变量文件
cp .env.example .env

# 2. 编辑 .env，添加你的 GA Measurement ID
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX

# 3. 重启开发服务器
npm run dev
```

### 2. 获取 GA Measurement ID

1. 访问 https://analytics.google.com/
2. 创建账号和媒体资源
3. 选择"网站"平台
4. 复制"衡量 ID"（格式：G-XXXXXXXXXX）

### 3. 验证安装

1. 访问网站
2. 打开浏览器开发者工具 -> Network
3. 查找 `google-analytics.com` 的请求
4. 或在 GA4 后台查看"实时"报告

## 📊 数据分析建议

### 关键指标

监控以下指标：
- **访问量**：每日/每周访问人数
- **转化率**：访问 → 上传 → 生成 → 导出
- **用户留存**：7天/30天留存率
- **热门页面**：哪些页面最受欢迎

### 转化漏斗分析

```
100% 访问首页
 ↓
 ?% 上传图片 (Step 1)
 ↓
 ?% 确认分析 (Step 2)
 ↓
 ?% 生成指南 (Step 3)
 ↓
 ?% 导出/分享 (Step 4)
```

找出流失最严重的环节，针对性优化。

# 小红书运营工具集 - 开发计划

## 📋 总体目标
将项目打造成一个完整的小红书运营工具平台，包含内容创作、数据分析、选题灵感等核心功能，为后续商业化做准备。

---

## 🎯 第一阶段：核心工具完善（优先级：高）

### ✅ 任务 1：热词洞察工具（整合概念词典）
**目标**：将"话题分析工具"升级为"热词洞察工具"，帮助用户发现热门概念和趋势词

**具体步骤**：
1. 查看 `gainiancidian` 项目的数据结构
   - 确认 `categories.json` 和 `concepts.json` 的数据格式
   - 了解概念数据的字段（标题、摘要、标签、分类等）

2. 创建数据适配层
   - 在 `src/data/` 下创建 `hotWordsData.ts`
   - 将概念词典数据转换为适合小红书选题的格式
   - 提取关键词、标签、分类信息

3. 创建工具页面
   - 创建 `src/views/tools/HotWordsInsightView.vue`
   - 功能模块：
     - 搜索框：搜索热门概念
     - 分类筛选：按领域筛选（心理学、经济学、营销等）
     - 热门榜单：展示热门概念TOP20
     - 趋势分析：展示最近新增/热门的概念
     - 相关推荐：基于当前概念推荐相关话题

4. 更新路由
   - 将 `/tools/topic-analyzer` 路由指向新页面
   - 更新 `ToolsView.vue` 中的工具描述

5. 测试和优化
   - 测试搜索、筛选功能
   - 优化界面展示
   - 添加使用说明

**预计时间**：2-3小时

---

### ✅ 任务 2：灵感话题库（整合人生第一次）
**目标**：将"话题分析工具"扩展为"灵感话题库"，提供"第一次XX"类选题灵感

**具体步骤**：
1. 查看 `firsttimeforlife` 项目的数据结构
   - 确认 `topics.js` 和 `taxonomy.json` 的数据格式
   - 了解话题数据的字段（标题、分类、标签、步骤等）

2. 创建数据适配层
   - 在 `src/data/` 下创建 `topicInspirationData.ts`
   - 将"人生第一次"数据转换为小红书选题格式
   - 提取可用的选题角度和关键词

3. 创建工具页面
   - 创建 `src/views/tools/TopicInspirationView.vue`
   - 功能模块：
     - 分类浏览：按分类查看话题（出行、财务、社交等）
     - 搜索功能：搜索关键词
     - 随机推荐：随机推荐3-5个话题
     - 话题详情：展示话题的完整信息（步骤、注意事项等）
     - 一键生成：点击话题后，直接跳转到"爆款生成器"并填充关键词

4. 更新路由和导航
   - 可以创建新路由 `/tools/topic-inspiration`
   - 或者在现有"话题分析工具"中增加"灵感话题"标签页

5. 测试和优化
   - 测试数据加载和展示
   - 优化话题推荐算法
   - 添加与"爆款生成器"的联动

**预计时间**：2-3小时

---

### ✅ 任务 3：完善现有工具的使用体验
**目标**：让每个工具都有清晰的使用指引和示例

**具体步骤**：
1. 为每个工具添加使用说明
   - 在工具页面顶部添加"使用说明"折叠面板
   - 包含：功能说明、使用步骤、注意事项

2. 添加示例数据
   - 为"标题生成器"添加示例输入
   - 为"爆款生成器"添加示例场景
   - 为图片工具添加示例图片

3. 优化交互流程
   - 添加"快速开始"按钮
   - 优化表单布局
   - 添加输入提示和验证

4. 添加结果导出功能
   - 标题生成器：支持复制、导出为文本
   - 爆款生成器：支持导出为Markdown或PDF

**预计时间**：1-2小时

## 🎯 第三阶段：变现准备（优先级：中高）

### ✅ 任务 6：设计付费功能点
**目标**：明确哪些功能免费，哪些功能付费

**具体步骤**：
1. 功能分级
   - **免费功能**：
     - 基础标题生成（每日10次）
     - 基础爆款生成（每日3次）
     - 图片背景移除（每日5次）
     - 基础内容浏览
   
   - **付费功能**：
     - 无限次使用所有工具
     - 高级数据分析工具
     - 批量处理功能
     - 优先客服支持
     - 专属内容库

2. 创建付费页面
   - 更新 `PricingView.vue`
   - 展示功能对比表
   - 添加支付入口

3. 添加使用限制
   - 在工具中添加使用次数限制
   - 添加"升级到Pro"提示
   - 添加使用统计

**预计时间**：3-4小时

## 🎯 第四阶段：高级功能开发（优先级：低）

### ✅ 任务 8：实现其他占位符工具
**目标**：将剩余的占位符工具逐步实现

**优先级排序**：
1. **SEO关键词工具**（高优先级）
   - 功能：关键词挖掘、搜索量分析、竞争度评估
   - 数据源：可以整合公开的SEO数据

2. **竞品分析工具**（中优先级）
   - 功能：账号分析、内容分析、增长追踪
   - 需要：数据源支持（可能需要API）

3. **图片编辑器**（中优先级）
   - 功能：在线图片编辑、模板库、滤镜效果
   - 需要：图片处理库（如fabric.js）

4. **定时发布工具**（低优先级）
   - 功能：定时发布、批量管理
   - 需要：后端支持

**预计时间**：每个工具 4-6小时

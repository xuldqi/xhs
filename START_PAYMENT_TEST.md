# 🚀 开始支付宝支付测试

## 快速开始（3 步完成测试）

### 步骤 1: 确认服务运行 ✅

后端服务已启动在 `http://localhost:3001`

检查状态：
```bash
curl http://localhost:3001/api/payment/check-config
```

### 步骤 2: 打开测试页面

**选择一种方式：**

#### 方式 A: 直接打开 HTML 文件
```bash
open test-payment-flow.html
```

#### 方式 B: 使用本地服务器（推荐）
```bash
python3 -m http.server 8000
```
然后访问: http://localhost:8000/test-payment-flow.html

### 步骤 3: 完成支付测试

1. **在测试页面中**
   - 点击"检查配置"（应该显示全部 ✅）
   - 输入测试金额: `0.01`
   - 点击"创建订单并跳转支付"

2. **在支付宝沙箱页面**
   - 使用沙箱买家账号登录
   - 确认支付 ¥0.01
   - 输入支付密码: `111111`

3. **验证结果**
   - 支付成功后会跳转回测试页面
   - 查看订单状态是否更新为"已支付"

## 📱 获取支付宝沙箱账号

访问: https://open.alipay.com/develop/sandbox/app

登录后可以看到：
- 买家账号（用于测试支付）
- 支付密码（通常是 111111）

## 🎯 测试目标

- [x] 配置检查通过

- [ ] 跳转到支付宝

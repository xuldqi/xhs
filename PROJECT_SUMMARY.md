## 🚀 快速开始

查看 `SETUP.md` 获取详细的安装和使用说明。

简要步骤：
1. `npm install` - 安装依赖
2. 配置 `.env` 文件（OpenAI API 密钥）
3. `npm run dev` - 启动开发服务器
4. 访问 http://localhost:5173
